# hangman-simple-in-C
## Simple hangman game in c 
* the game has 3 stage.
* before each stage player 1 needs to enter the gussed word and player 2 will guess the ward.
* if player 2 get through the first stage he/she will be given an extra life which can be used in case of wromg guess in second or third stage.
* before each stage player is saked whether they want to continue or not.
* when the player successfully end all three stage a success message will be shown.
* the game will end when player used up all the lives or when successfully finished all stage.
